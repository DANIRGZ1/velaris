Policy Compliance

Built within Riot's rules.

Velaris is designed to comply with Riot Games' Third-Party Application Policy. Here's exactly what we do and don't do.

✅
Official APIs only
All data is fetched through the official Riot Games REST API (MATCH-V5, SUMMONER-V4, LEAGUE-V4) and the Live Client Data API provided at localhost:2999. No game memory is read. Velaris is fully compatible with Vanguard.
✅
No automation of gameplay
Velaris does not automate any in-game action. The "match found" notification shows a toast with an Accept button that the user must actively click. No scripts, no bots.
✅
No competitive advantage via hidden information
The in-game overlay does not track enemy summoner spell cooldowns, show level-6 alerts for specific champions, or expose any information Riot intentionally withholds from players. Objective timers are derived from public game events (DragonKill, BaronKill) visible to all players.
✅
Rate limits respected
All Riot API calls respect the published rate limits. Match history is cached locally to minimize API calls. Polling intervals are set to reasonable durations.
